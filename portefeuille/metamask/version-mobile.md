# Version Mobile

La version mobile de MetaMask est à la fois un portefeuille **et un navigateur**.&#x20;

Elle est disponible gratuitement pour IOS ou Android.

Pour accéder à une application Web3, qui nécessitera une identification avec votre wallet, vous devez :

* Aller dans la partie navigateur de MetaMask (second icone en bas), par défaut le navigateur s'ouvre sur l'adresse portfolio.metamask.io
* Tout en haut, vous pouvez : entrer l'url de la DApp (ici RMM), puis vous connectez avec votre wallet,
* Si vous avez plusieurs comptes, et que ce n'est pas le bon qui se connecte, cliquez en haut à droite, puis choisissez le bon compte :

<figure><img src="../../.gitbook/assets/image (340).png" alt=""><figcaption></figcaption></figure>

* Vous pouvez enregistrer en tant que favoris, vos DApp habituelles : via les trois petits points en bas à droite. Vous retrouverez vos favoris en cliquant sur la barre d'URL (en haut).

<figure><img src="../../.gitbook/assets/image (342).png" alt="" width="527"><figcaption></figcaption></figure>
