# Ajout d'un réseau

MetaMask est installé avec plusieurs réseaux (blockchain) par défaut (Ethereum, Polygon,..), mais pas Gnosis ! \
RealT est disponible sur Ethereum et Gnosis, ce dernier étant préférable pour limiter les frais de transactions.

## Pour ajouter le réseau Gnosis :

*   Accédez au site Web de [ChainList](https://chainlist.org/), connectez votre wallet MetaMask, recherchez Gnosis et l'ajouter :

    <figure><img src="../../.gitbook/assets/image.png" alt="" width="485"><figcaption></figcaption></figure>
*   Vous pourrez visualiser l'ensemble des caractéristiques du réseau (dont le [RPC](modification-du-rpc.md)), avec la fonction "Modifier" du réseau :

    <figure><img src="../../.gitbook/assets/image (1).png" alt="" width="506"><figcaption></figcaption></figure>

Avec le site web ci-dessus, la méthode est très simple et automatique. Vous pouvez aussi faire cet ajout plus "manuellement" dans MetaMask : avec la fonction ajout de réseau de MetaMask, puis compléter des infos demandées avec celles figurant dans l'image ci-dessus.
